SARS-CoV-2 genome networks

Building upon the K-mer signature analysis we performed, we constructed a network representing the diversity among the SARS-CoV-2 genomes. Each point represents a single strain and the distance between them is the calculated euclidian distance, based on the k-mer signatures. By setting the threshold to filter out more distant relationships, we are able to investigate how the different strains are connected to each other.
